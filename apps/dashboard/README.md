# Forge Dashboard

> **Ships in v1.1** — Coming soon!

The Forge Dashboard is a Next.js 14 web application for monitoring and managing Forge executions.

## Planned Features (v1.1)

- 🖥️ Active executions overview
- 📋 Execution detail view (task list, logs, token usage)
- ⚙️ Settings (configure API endpoint)
- 🌙 Dark mode default

## Contributing

Interested in building the dashboard? Check out the [roadmap](../../docs/ROADMAP.md) and open a PR!

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
