# Forge VS Code Extension

> **Ships in v1.2** — Coming soon!

The Forge VS Code Extension lets you run Forge goals directly from your editor.

## Planned Commands (v1.2)

| Command | Description |
|---|---|
| `Forge: Run Goal` | Prompt for goal and execute via Forge API |
| `Forge: Show Status` | Display execution status in output panel |
| `Forge: Show Logs` | Stream execution logs |
