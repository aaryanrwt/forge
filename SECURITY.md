# Security Policy

## Supported Versions

We actively support and apply security patches to the following versions of Forge:

| Version | Supported |
|---|---|
| **v1.0.x** (Community Edition) | ✅ Yes |
| **< v1.0.0** | ❌ No |

## Reporting a Vulnerability

We take the security of Forge seriously. If you discover any security vulnerabilities, please do **not** open a public issue. Instead, report it privately to our security team.

### How to Report
Please email your report to **security@forge-ai.org** with the following details:
- **Description**: A detailed description of the vulnerability.
- **PoC**: Step-by-step instructions or proof-of-concept code to reproduce the issue.
- **Impact**: The potential impact of the vulnerability if exploited.

We will acknowledge receipt of your report within **48 hours** and provide a tracking ID and estimated timeline for a patch.
